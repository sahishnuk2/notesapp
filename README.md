# NoteApp (Go + Gin + SQLite)

A small practice project to learn Golang, Gin, and SQLite.
Supports creating, listing, deleting, and sharing notes using simple HTML templates.

### Tech Stack

- Go (Golang)
- Gin Web Framework
- SQLite
- HTML Templates

### Run
```
go run main.go
```


### Then open:

```
http://localhost:8080
```

### Notes
Built purely for learning Go + database basics
